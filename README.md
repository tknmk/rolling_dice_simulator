# probability_player3_win.py

```
Enter Player A dice value: 3
Enter Player A dice value: 5
1 / 6
```

# permutation_combination.py

順列と組み合わせを出力します。

```
[('a', 'b'), ('a', 'c'), ('b', 'a'), ('b', 'c'), ('c', 'a'), ('c', 'b')]
6
[('a', 'b'), ('a', 'c'), ('b', 'c')]
3
```
