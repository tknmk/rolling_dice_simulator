# word_balancer.py 

正となる単語を1つ指定し、類似する(表記揺れ)単語をリストでもたせる。
正となる単語を元に、類似している単語の類似率を算出し、出力する。

```
正となる単語: サーバ
[('サーバー', '85.71%'),
 ('サーバ', '100.00%'),
 ('ｻｰﾊﾞｰ', '85.71%'),
 ('ｻｰﾊﾞ', '100.00%'),
 ('ドッカー', '28.57%')]
```
